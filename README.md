# Wedding Digital Invitation

Undangan digital pernikahan Adi Dwi Saputra & Zahro Annisa berbasis Vue 3 + Vite.

## Menjalankan Project

```bash
npm install
npm run dev
```

Server development akan tersedia di `http://localhost:5173/`.

## Environment

Salin `.env.example` menjadi `.env`, lalu isi konfigurasi Supabase:

```bash
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-public-anon-key
```

Jika Supabase project memakai publishable key baru, gunakan `VITE_SUPABASE_PUBLISHABLE_KEY`.

## Prinsip Desain

Modern, Elegan, Syar'i, Bermakna, dan Tidak Berlebihan.
